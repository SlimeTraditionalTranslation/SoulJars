# 靈魂儲存罐
黏液科技的附加 繁體翻譯版

靈魂儲存罐是黏液科技的附加插件,其靈感來自於著名的模組[SoulShards](https://www.curseforge.com/minecraft/mc-mods/soul-shards-respawn).
它添加了新物品名為"靈魂罐"
與模組類似,可以捕捉生物靈魂.
你可以使這些靈魂罐頭製作已損壞的生怪磚,對其進行修復,從而變成任何你喜歡的生物生怪磚.
該附家具有設定來啟用/禁用任何靈魂罐. 因此請查看靈魂罐的config.yml.
